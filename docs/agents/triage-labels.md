# Triage Labels

The skills speak in terms of canonical triage roles. This repo only uses the
subset below. Do not create or apply missing triage labels unless the user asks
for them in the current task.

| Label in mattpocock/skills | Label in our tracker | Meaning                         |
| -------------------------- | -------------------- | ------------------------------- |
| `ready-for-agent`          | `ready-for-agent`    | Fully specified, AFK-ready      |
| `ready-for-human`          | `ready-for-human`    | Needs a human decision or pass  |
| `wontfix`                  | `wontfix`            | Will not be actioned            |

## Labels not used here

- `needs-triage`
- `needs-info`

If a skill asks for one of the unused roles, leave the issue unlabeled for that
role and explain the repo does not use that label vocabulary.
